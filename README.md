# 🔮 Mysterious AI Suite (Ubuntu)

A simple modular AI installer for Ubuntu.

## Features
- One-command launcher
- Install ComfyUI, AUTOMATIC1111, InvokeAI
- Auto-update existing installs
- Beginner-friendly

## Setup

git clone <your-repo>
cd mysterious-ai-suite
chmod +x launch.sh
./launch.sh

## Apps Included
- ComfyUI
- AUTOMATIC1111 Stable Diffusion
- InvokeAI

## Notes
Each app installs in:
~/mysterious/apps/

## Goal
Provide a clean, simple, and extendable AI setup system.
